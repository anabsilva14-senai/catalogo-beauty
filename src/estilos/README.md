# 🎨 Estilos

Esta pasta contém os arquivos de estilização global da aplicação
Beauty Collective Cosmetics.

## Arquivos presentes:

- **App.css** → Estilos globais da aplicação, incluindo cores,
  fontes e layout geral
- **index.css** → Estilos base do projeto, reset de CSS e
  configurações iniciais

## 🖌️ Paleta de Cores do Projeto:

| Nome        | Cor       | Uso                        |
|-------------|-----------|----------------------------|
| Rosa principal | #F4A7B9 | Botões e destaques        |
| Branco      | #FFFFFF   | Fundo das páginas          |
| Cinza claro | #F5F5F5   | Cards e seções             |
| Preto suave | #333333   | Textos e títulos           |

## 📐 Padrões de estilo adotados:

- Fontes: **Poppins** e **Playfair Display**
- Layout baseado em **Flexbox** e **Grid**
- Design **responsivo** para mobile e desktop
- Variáveis CSS utilizadas para manter consistência visual
