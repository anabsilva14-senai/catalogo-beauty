# 📦 Componentes

Esta pasta contém todos os componentes reutilizáveis da interface do projeto Beauty Collective Cosmetics.

## Arquivos presentes:
- **CartaoContato.jsx** → Card com informações de contato
- **CartaoProduto.jsx** → Card de exibição de produto
- **CartaoPromocao.jsx** → Card de promoções e ofertas
- **slideshow.jsx** → Carrossel de imagens da loja
- **Topo.jsx** → Cabeçalho/Header do site
- **Rodape.jsx** → Rodapé/Footer do site
